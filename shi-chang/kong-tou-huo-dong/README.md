# 空投活动

为了不断地进行市场推广，ZGoat产品在设计之初就考虑了诸多市场推广方案，包括：心愿单、空投、社群活动等等。空投的ZGoat由运营账号提供，各个社群的群主如果有意合作，可联系商务负责人进行对接。

合作基本要求：

* 社群为加密行业社群，电报人数&gt;3000人，或推特关注量&gt;3w人，或微信群&gt;300人
* 空投形式的参与门槛低
* 具备一定的娱乐性

商务联系方式：

* 电报：@zgoat\_org
* 微信：zgoat\_org
